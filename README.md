# HabitFlow-portfolio
架空SaaS習慣トラッカーLP（ミニマルデザイン）

架空の習慣トラッカーアプリのランディングページ。
ミニマル・シンプルデザインで、20〜30代ビジネスパーソン向けに「続けやすい」UIを目指しました。

- 雰囲気：黒背景 + グラデーションアクセント + 余白多め
- 機能：1日1習慣、ストリーク可視化、リマインダーオフ可能
- 技術：HTML/CSS（変数活用）、スマホファースト
- ベース：Claude生成 → 自分で色調調整・UI微改善を実装

デモ：https://ユーザー名.github.io/habitflow-lp/ （GitHub Pages設定後）
